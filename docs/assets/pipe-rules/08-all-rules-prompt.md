## 7种规则确认

**双通（3种）**
1. 同径弯头 → 平分
2. 变径直通 → 全归大口径
3. 弯头+变径 → 变径归连接管道

**三通（2种）**
4. 三通本体 → 三等分
5. 三通口接变径 → 变径归连接管道

**四通（2种）**
6. 四通本体 → 四等分
7. 四通口接变径 → 变径归连接管道
