## 📍 定位说明

### 与标准 4 层架构的关系

```
docs/
├── ai-context/      # 第 1 层：AI 上下文（项目特定）
├── development/     # 第 2 层：开发文档（项目特定）
├── architecture/    # 第 3 层：架构文档（项目特定）
├── project/         # 第 4 层：项目文档（项目特定）
└── guides/          # ⚠️ 参考文档（通用，不属于 4 层）
```

### 使用场景

1. **项目初始化**：参考 `01-AI协作系统配置指南.md` 建立文档体系
2. **Slash Commands 编写**：参考 `02-Slash-Commands完整配置.md` 的标准模板
3. **系统排查**：参考 `03-系统健康排查指南.md` 诊断问题

### 注意事项

- 这些文档是**参考标准**，不应被 `/start` 等命令自动读取
- 项目特定的配置应在 `ai-context/CONTEXT.md` 中记录
- 如需修改 Slash Commands，应直接编辑 `.claude/commands/` 下的文件
