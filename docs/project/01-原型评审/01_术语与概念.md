# 术语与概念

## 一、Revit 术语

### 1.1 族 (Family)

Revit 中的参数化构件模板。所有添加到三维模型中的结构构件（如墙体、门窗、设备等）都是基于族创建的。

**族的分类**：
- **系统族**：Revit 内置，如墙、楼板、屋顶等
- **可载入族（自建族）**：独立 .rfa 文件，如门、窗、设备等
- **内建族**：项目内特定的自定义构件

### 1.2 族类型 (Family Type)

族的不同变体，通过参数值区分。

**示例**：
- 族：新风机
- 族类型：新风机-3000M3/H、新风机-5000M3/H、新风机-7000M3/H

### 1.3 族实例 (Family Instance)

在模型中实际放置的构件，是族类型的具体应用。

**示例**：模型中放置了 3 个「新风机-3000M3/H」，则有 3 个族实例。

### 1.4 参数类型

| 类型 | 说明 | 修改影响 |
|-----|------|---------|
| **类型参数** | 属于族类型，同类型共享 | 修改影响所有同类型实例 |
| **实例参数** | 属于族实例，各实例独立 | 修改只影响当前实例 |

## 二、系统术语（YBP）

### 2.1 术语对应关系

| Revit 术语 | 系统术语 | 说明 |
|-----------|---------|------|
| 族 | **品目** | 一类构件的模板 |
| 族类型 | **品目类型** / **品目实例** | 品目的不同变体 |
| 族实例 | - | 模型中实际使用的构件 |

### 2.2 品目 (Item)

系统中管理的族，对应 Revit 中的族。

**品目层级包含**：
- 关联清单（该品目可能匹配的所有清单）
- 品目参数（构件属性 + 工程量属性）
- 品目类型（预设的族类型）

### 2.3 品目参数

在品目层级定义的参数，分为两类：

| 类型 | 用途 | 说明 |
|-----|------|------|
| **构件属性** | 匹配清单 | 定义用于清单匹配的参数和规则 |
| **工程量属性** | 计算工程量 | 定义工程量的计算方式 |

## 三、业务术语

### 3.1 清单

工程量清单项，按照国家规范编制，用于招标投标和工程计价。

**清单要素**：
- 项目编码
- 项目名称
- 项目特征
- 计量单位
- 工程量

**示例**：
- 新风机3000M3/H
- 防雨排风百叶(截面≤0.25m²)
- 150x150mm灰色砖地面（琛雄，含不超过50mm厚的找平层）

### 3.2 工程量

建筑工程中各分部分项工程的实物数量。

**计量方式**：
- 计数：台、个、套
- 面积：m²
- 长度：m
- 体积：m³

### 3.3 匹配条件

每个清单配置的规则，用于判断族实例是否应该带出该清单。

**示例**：
- 风量 >= 4000
- 结构材质 = "厨房灰色砖" AND 品牌 = "琛雄"
