# YBP 网页设计总览

> GitHub Dark 风格的温暖深色主题设计方案

**创建日期**: 2025-11-25
**最后更新**: 2025-12-10
**设计风格**: GitHub Dark 温暖深色调 + 侧边导航

---

## 一、设计背景

### 1.1 项目现状

- 需求讨论完成
- 原型设计完成
- 网页展示系统开发完成
- 易达评审中，准备开发

### 1.2 网页目标

| 目标 | 说明 | 优先级 |
|-----|------|--------|
| **汇报演示** | 向领导展示项目价值和进展 | P0 |
| **会议讨论** | 与易达讨论时的可视化参考 | P0 |
| **知识沉淀** | 结构化保存讨论成果 | P1 |

### 1.3 设计原则

- **温暖**: 深色调但不纯黑，带温暖感
- **专业**: 参考 GitHub Dark 设计语言
- **可扩展**: 侧边导航支持未来功能扩展

---

## 二、页面架构

### 2.1 页面结构

```
YBP 网页
│
├── 概览 (/)
│   ├── 统计卡片（4 个指标）
│   ├── 快速入口网格
│   ├── 最近活动列表
│   ├── 核心流程图
│   └── 项目阶段进度
│
├── 业务逻辑 (/logic)
│   ├── 核心流程 badge
│   ├── 5 种匹配规则卡片
│   └── 4 种工程量计算类型
│
├── 工程量梳理 (/quantity/*)
│   ├── /quantity - 总览
│   ├── /quantity/owner - 业主清单
│   ├── /quantity/pipe - 水管系统
│   └── /quantity/duct - 风管系统
│
└── 问题追踪 (/issues)
    ├── 统计卡片（可点击筛选）
    ├── 筛选标签栏
    └── 问题列表
```

### 2.2 路由配置

| 页面 | 路由 | 组件 | 说明 |
|-----|------|------|------|
| 概览 | `/` | HomePage | 仪表盘风格首页 |
| 业务逻辑 | `/logic` | LogicPage | 匹配规则 + 计算类型 |
| 工程量梳理 | `/quantity/*` | QuantityPage | 12月核心任务 |
| 问题追踪 | `/issues` | IssuesPage | 问题跟踪与筛选 |

## 三、布局系统

### 3.1 整体布局

```
┌─────────────────────────────────────────────────────────┐
│                     App Layout                          │
├──────────┬──────────────────────────────────────────────┤
│          │                                              │
│  Sidebar │              Main Content                    │
│  (260px) │                                              │
│          │  ┌────────────────────────────────────────┐  │
│  - 概览   │  │            Top Bar (64px)              │  │
│  - 业务   │  ├────────────────────────────────────────┤  │
│  - 工程量 │  │                                        │  │
│    ├ 业主 │  │            Page Content                │  │
│    ├ 水管 │  │                                        │  │
│    └ 风管 │  │                                        │  │
│  - 问题   │  │                                        │  │
│          │  └────────────────────────────────────────┘  │
│          │                                              │
└──────────┴──────────────────────────────────────────────┘
```

### 3.2 侧边导航

- **展开宽度**: 260px
- **折叠宽度**: 72px
- **移动端**: 抽屉式（点击遮罩关闭）
- **子菜单**: 工程量梳理支持展开子菜单

### 3.3 顶部栏

- **高度**: 64px
- **内容**: 页面标题 + 搜索 + 通知（预留）
- **固定定位**: sticky top-0

## 四、视觉设计系统

### 4.1 设计风格

- **整体**: GitHub Dark 温暖深色调
- **参考**: GitHub、Linear、Vercel Dashboard
- **特点**: 层次渐进背景、柔和边框、状态指示

### 4.2 配色方案

```css
/* 背景色（层次渐进，非纯黑）*/
--bg-base:      #0d1117;  /* 最深背景 */
--bg-surface:   #161b22;  /* 主表面 */
--bg-elevated:  #1c2128;  /* 卡片背景 */
--bg-hover:     #262c36;  /* 悬浮状态 */
--bg-active:    #2d333b;  /* 激活状态 */

/* 边框色 */
--border:       #30363d;  /* 默认边框 */
--border-muted: #21262d;  /* 柔和边框 */

/* 文字色 */
--text-primary:   #e6edf3;  /* 主文字 */
--text-secondary: #8b949e;  /* 次要文字 */
--text-tertiary:  #6e7681;  /* 辅助文字 */
--text-link:      #58a6ff;  /* 链接 */

/* 强调色 */
--accent-blue:   #58a6ff;
--accent-green:  #3fb950;
--accent-amber:  #d29922;
--accent-red:    #f85149;
--accent-purple: #a371f7;
```

### 4.3 字体规范

```css
font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI',
             'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei',
             'Helvetica Neue', Helvetica, Arial, sans-serif;
```

**系统字体栈**，优先使用系统原生字体，保证跨平台一致性。

### 4.4 圆角规范

```
├── rounded-md:  6px   // 小元素（badge）
├── rounded-lg:  8px   // 输入框、按钮
├── rounded-xl:  12px  // 卡片
└── rounded-2xl: 16px  // 大卡片、导航项
```

## 五、组件类

### 5.1 卡片 (.card)

```css
.card {
  background: var(--bg-elevated);
  border: 1px solid var(--border-muted);
  border-radius: 12px;
  transition: all 0.2s;
}
.card:hover {
  border-color: var(--border);
}
```

### 5.2 徽章 (.badge)

```css
.badge {
  padding: 2px 8px;
  border-radius: 6px;
  font-size: 12px;
  font-weight: 500;
}

/* 变体 */
.badge-blue   { background: rgba(88,166,255,0.15); color: #58a6ff; }
.badge-green  { background: rgba(63,185,80,0.15);  color: #3fb950; }
.badge-amber  { background: rgba(210,153,34,0.15); color: #d29922; }
.badge-red    { background: rgba(248,81,73,0.15);  color: #f85149; }
.badge-purple { background: rgba(163,113,247,0.15); color: #a371f7; }
```

### 5.3 状态点 (.status-dot)

```css
.status-dot {
  width: 8px;
  height: 8px;
  border-radius: 50%;
}
.status-dot.success { background: var(--accent-green); }
.status-dot.warning { background: var(--accent-amber); }
.status-dot.info    { background: var(--accent-blue); }
```

### 5.4 按钮 (.btn)

```css
.btn-ghost {
  background: transparent;
  color: var(--text-secondary);
  padding: 8px 12px;
  border-radius: 8px;
}
.btn-ghost:hover {
  background: var(--bg-hover);
  color: var(--text-primary);
}
```

## 六、动画系统

### 6.1 入场动画

```css
@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}

@keyframes fadeUp {
  from { opacity: 0; transform: translateY(10px); }
  to { opacity: 1; transform: translateY(0); }
}

.animate-fade-in { animation: fadeIn 0.3s ease-out forwards; }
.animate-fade-up { animation: fadeUp 0.4s ease-out forwards; }
```

### 6.2 交互过渡

```css
/* 所有交互元素 */
transition: all 0.2s ease;

/* 侧边导航展开/折叠 */
transition: width 0.3s ease, transform 0.3s ease;
```

## 八、响应式设计

