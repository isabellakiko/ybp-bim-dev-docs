# 问题追踪

> 项目问题管理与待讨论会议

---

## 目录结构

```
issues/
├── open/        # 待解决问题
├── resolved/    # 已解决问题
└── meetings/    # 待讨论会议（已结论的会议归档到各阶段）
```

---

## 问题状态

| 状态 | 说明 | 目录 |
|------|------|------|
| 待讨论 | 需要进一步讨论确认 | open/ |
| 进行中 | 正在处理 | open/ |
| 已解决 | 已有明确结论 | resolved/ |
