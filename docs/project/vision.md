# YBP 项目愿景

## 项目概述

**项目名称**：YBP (永麦 BIM Platform)

**需求方**：上海永麦管理咨询有限公司

**开发方**：广州易达建信科技开发有限公司

## 核心理念

### 建模即算量

YBP 的核心理念是"建模即算量"，通过 Revit 模型自动计算工程量，彻底改变传统的工程造价计算方式。

## 业务价值

### 传统方式的痛点

- **效率低下**：基于 CAD 图纸手动拉线计算工程量
- **容易出错**：人工操作带来的计算错误和遗漏
- **重复劳动**：设计变更后需要重新计算
- **信息割裂**：设计模型与工程量计算分离

### YBP 的解决方案

通过 BIM 技术实现从设计模型到工程量的自动化转换：

1. **自动化**：基于 Revit 模型自动提取和计算工程量
2. **准确性**：减少人工干预，提高计算精度
3. **实时性**：模型变更后工程量自动更新
4. **一体化**：打通设计与造价的信息流

## 业务流程

YBP 遵循标准的工程造价业务流程：

```
族(Family) → 品目 → 清单 → 工程量
```

### 流程说明

1. **族 (Family)**
   - Revit 族是 BIM 模型的基本构建单元
   - 包含构件的几何信息、属性参数等

2. **品目**
   - 将 Revit 族映射到造价品目
   - 建立 BIM 构件与造价科目的对应关系

3. **清单**
   - 按照工程清单规范组织品目
   - 形成标准的工程量清单结构

4. **工程量**
   - 基于 Revit 模型自动计算各品目的工程量
   - 生成详细的工程量计算书

## 技术愿景

### 平台化

- 构建可扩展的 BIM 工程量计算平台
- 支持多项目、多用户协同工作

### 智能化

- 利用 AI 技术优化族-品目映射
- 自动识别和推荐品目配置

### 标准化

- 遵循国家工程量清单规范
- 建立标准的族库和品目库

## 目标用户

### 主要用户

- **造价工程师**：进行工程量计算和造价分析
- **BIM 工程师**：维护 Revit 模型和族库
- **项目经理**：查看工程量和成本报表

### 使用场景

- 施工图阶段的工程量计算
- 设计变更的工程量对比
- 成本控制和分析

## 成功标准

### 业务目标

- 工程量计算效率提升 80% 以上
- 计算准确率达到 99% 以上
- 支持 100+ 个并发项目

### 技术目标

- 系统响应时间 < 2 秒
- 大型项目（10000+ 构件）计算时间 < 5 分钟
- 系统可用性 > 99.9%
