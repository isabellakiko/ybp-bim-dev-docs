# 首页设计

> 项目概览页，让访问者快速了解 YBP 项目

**最后更新**: 2025-11-27

---

## 一、页面目标

| 目标 | 说明 |
|-----|------|
| **30秒了解** | 项目是什么、解决什么问题 |
| **1分钟理解** | 核心业务流程 |
| **2分钟掌握** | 项目进展、下一步 |

---

## 二、页面结构

```
首页 (/)
├── Hero Section（英雄区）
│   ├── 状态标签（脉冲动画）
│   ├── 项目标题 YBP
│   ├── 副标题：族库管理系统
│   ├── 描述文字
│   └── CTA 按钮组
│
├── Flow Section（核心流程）
│   └── 四步流程图（渐变卡片）
│       族 → 品目 → 清单 → 工程量
│
├── Progress Section（项目进度）
│   └── 时间线（带动画进度条）
│
└── Quick Access Section（快速入口）
    ├── 业务逻辑卡片
    └── 问题追踪卡片
```

## 三、各区块设计

### 3.1 Hero Section

**布局**: 居中对齐，渐变背景

**内容**:
- 状态标签: `易达评审中`（带脉冲动画圆点）
- 标题: `YBP`（5xl-7xl 响应式）
- 副标题: `族库管理系统`
- 描述: 基于 BIM 模型的智能清单匹配与工程量自动计算
- CTA: `了解业务逻辑` + `查看问题追踪`

**动画**: animate-fade-up 依次入场

### 3.2 Flow Section（核心流程）

**布局**: 水平流程图，带箭头连接

```
┌────────────┐     ┌────────────┐     ┌────────────┐     ┌────────────┐
│    族      │ ──▶ │   品目     │ ──▶ │   清单     │ ──▶ │  工程量    │
│  Family    │     │   Item     │     │   Bill     │     │ Quantity   │
│ 蓝色渐变   │     │ 紫色渐变   │     │ 绿色渐变   │     │ 橙色渐变   │
└────────────┘     └────────────┘     └────────────┘     └────────────┘
```

**样式**:
- 卡片: 24-28px 方形，圆角 3xl
- 渐变: bg-gradient-to-br
- 悬浮: hover-lift 效果
- 箭头: 灰色 SVG，sm:block

**数据**:
```javascript
const flowNodes = [
  { id: 'family', label: '族', sub: 'Family', color: 'from-blue-500 to-blue-600' },
  { id: 'item', label: '品目', sub: 'Item', color: 'from-purple-500 to-purple-600' },
  { id: 'bill', label: '清单', sub: 'Bill', color: 'from-emerald-500 to-emerald-600' },
  { id: 'quantity', label: '工程量', sub: 'Quantity', color: 'from-amber-500 to-amber-600' },
]
```

### 3.3 Progress Section（项目进度）

**布局**: 水平时间线，带动画进度条

```
    ●────────●────────●────────◐────────○
 需求讨论  原型设计  易达评审  开发启动
  (完成)    (完成)   (进行中)   (待开始)
```

**样式**:
- 完成: 蓝色实心圆 + 对勾图标
- 进行中: 空心圆 + 脉冲动画圆点
- 待开始: 灰色空心圆
- 进度条: 蓝色，width 动态计算（66%）

**数据**:
```javascript
const milestones = [
  { label: '需求讨论', done: true },
  { label: '原型设计', done: true },
  { label: '易达评审', current: true },
  { label: '开发启动', done: false },
]
```

### 3.4 Quick Access Section（快速入口）

**布局**: 2 列卡片网格

**卡片设计**:
- 渐变图标（12x12，圆角 2xl）
- 标题 + 描述
- 悬浮效果: hover-lift + border 变色
- 点击跳转对应页面

**数据**:

## 四、动画效果

### 4.1 入场动画

```jsx
// Hero 区域使用 animate-fade-up
<div className="animate-fade-up" style={{ animationDelay: '0.1s' }}>
  ...
</div>
```

### 4.2 滚动动画

```jsx
// 使用 useScrollAnimation Hook
function useScrollAnimation() {
  const ref = useRef(null)

  useEffect(() => {
    const observer = new IntersectionObserver(...)
    const elements = ref.current?.querySelectorAll('.animate-on-scroll')
    elements?.forEach((el) => observer.observe(el))
    return () => observer.disconnect()
  }, [])

  return ref
}
```

### 4.3 交互动画

- 流程卡片: hover-lift
- 快速入口: hover:border-primary/50
- 状态标签: animate-pulse
